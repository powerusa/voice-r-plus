# Voice R+

Voice R+ is a local-first iOS voice recorder built with SwiftUI and AVFoundation.

## Features

- Record, pause, resume, stop, and save audio
- Save recordings locally as `.m4a` files
- Play recordings with progress, skip forward, and skip back controls
- Rename, delete, share, and export recordings
- Local metadata storage
- Microphone permission handling
- Modern SwiftUI interface with dark and light mode support

## Privacy

Voice R+ does not collect user data. Recordings stay on the user's device unless the user chooses to share or export them.

Privacy policy:
https://powerusa.github.io/voice-r-plus-privacy/

## Requirements

- iOS 17+
- Xcode 16 or newer
- SwiftUI
- AVFoundation
